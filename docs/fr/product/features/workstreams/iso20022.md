# Workstream Affinage ISO 20022

Ce workstream affine l’implémentation ISO 20022 livrée avec Mojaloop V17.0.0 à partir du retour des adoptants, y compris les messages ISO 20022 définis dans le *Market Practice Document* (MPD) et les mises à jour du MPD lui-même.

# Justification métier

ISO 20022 devient rapidement la norme mondiale de messagerie pour les systèmes de paiement ; les messages SIIP Mojaloop en ISO 20022, dont l’accord sur les conditions (*Agreement of Terms*), s’inscrivent dans ce mouvement. Les détails ne sont toutefois pas encore figés et des affinements restent probables.
