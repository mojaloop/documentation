# Catalogue des APIs

La famille des APIs Mojaloop regroupe plusieurs APIs différentes qui répondent à divers besoins métier ou transactionnels. Jusqu'à présent, les APIs bien définies et adoptées sont :

- [APIs d’Interopérabilité FSP (FSPIOP)](./fspiop/)
- [API d’Administration](./administration/)
- [API de Règlement](./settlement/)
- [API d’Initiation de Paiement Tiers (3PPI/PISP)](./thirdparty/)

D’autres APIs sont soit en cours de développement et de conception, soit prévues sur la feuille de route :

- API inter-réseaux (FX)
- API de reporting
