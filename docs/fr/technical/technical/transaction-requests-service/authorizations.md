# Demandes de transaction — Autorisations

GET /authorizations/{TransactionRequestID} et PUT /authorizations/{TransactionRequestID} pour prendre en charge les autorisations dans la « demande de paiement du marchand » (*Merchant Request to Pay*) et d'autres cas d'usage initiés par le Bénéficiaire.
