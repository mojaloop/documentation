# ISO 20022 Refinement Workstream
This workstream focusses on refining the ISO 20022 implementation released with Mojaloop V17.0.0 based on the experiences of adopters. This includes any refinements to the ISO 20022 messages defined in the Market Practice Document (MPD), and updates to the MPD itself.
